Jervice Documentation

Styled documentation for your mobile assistant and setup.

Setup

Install the Python backend and set the environment key for Jervice.

  1. Install dependencies: 
    pip install -r requirements.txt
  2. Set the Jervice API key: 
    set JERVICE_API_KEY="your_api_key_here"

    For PowerShell, use:

    $env:JERVICE_API_KEY = "your_api_key_here"

    For macOS/Linux, use:

    export JERVICE_API_KEY="your_api_key_here"
  3. Run the mobile backend: 
    python app.py
  4. Open the mobile page from your phone:
    5. http://YOUR-PC-IP:5000

Using Jervice

On the mobile page, type your question or tap the voice button to speak. Jervice will reply in text and speak the answer aloud.

Troubleshooting

Server startup

  • Install Flask and google-genai.
  • Set JERVICE_API_KEY before starting the app.
  • Allow port 5000 through your firewall.

Network error

  • Ensure your phone and PC are on the same Wi-Fi network.
  • Use your PC's local IP address, not localhost.
  • Confirm the backend is running.

Notes

  • Jervice is the assistant name; there is no Gemini branding in the UI.
  • Your API key stays on the server and is not exposed to the browser.
  • Open the live docs from any device on your network at /docs.
